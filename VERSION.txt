🚀 DEPLOYMENT
=============

Supported Platforms:
  - ✅ Vercel (recommended)
  - ✅ Netlify
  - ✅ Self-hosted

  - NEXT_PUBLIC_SITE_URL
  - OPENAI_API_KEY (for AI features)

Build Command:
  npm run build

Start Command:
  npm start

Dev Command:
  npm run dev
